Pour lancer le programme il y a plusieurs manières de le faire :
1/
	- nettoyer le dossier avec la commande "make clear"
	- compiler avec "make"
	- executer avec "make compute"
2/ lancer le script script_agprojet : "bash script_agprojet"
Le script automatise toutes les tâches précédentes.
Il n'est à lancer que sur Linux.
Le résultat de l'éxécution est écrite dans un fichier output.txt qui sera ouvert à la fin du programme.

Dans le programme il y a 2 exemples qui vous sont donnés afin de tester certaines fonctionnalités.
Ces exemples sont placés dans des commentaires /*...*/.
